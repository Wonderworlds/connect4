## Installation

git clone connect4 with git

```bash
git clone https://github.com/Wonderworlds/connect4.git

cd connect4

make
or
make bonus (for grapgical connect4)

./connect4 [height] [Width]
or
./connect4_bonus [height] [Width] [--display]

```
and you can play.
